# Understanding User Roles in Salesforce CRM

## Overview
Salesforce CRM uses role-based access to control what users can view and modify within the system.

## Common User Roles
- Sales Representative: Manages leads and opportunities
- Sales Manager: Oversees team performance and reports
- Support Agent: Handles customer-related activities
- Administrator: Manages system configuration and users

## Why Roles Matter
User roles help ensure data security and proper access to CRM features.
