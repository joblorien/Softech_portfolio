# Getting Started with Salesforce CRM

## Overview
This section helps new users begin working with Salesforce CRM by explaining basic access and navigation.

## Logging In
1. Open the Salesforce CRM login page.
2. Enter your organization email and password.
3. Click Log In to access the dashboard.

## Navigating the Interface
- Use the navigation bar to access modules such as Leads, Accounts, and Opportunities.
- The home page displays recent activities and performance metrics.
